Переводчик текста с помощью ИИ

Этот проект использует модель Helsinki-NLP/opus-mt-tc-bible-big-mul-mul для перевода текстов.

Модель Helsinki-NLP/opus-mt-tc-bible-big-mul-mul - это большая модель машинного обучения, предназначенная для перевода текстов на множество языков. Она обучена на большом объеме данных и может справиться с переводом сложных текстов.
